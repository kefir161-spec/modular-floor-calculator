# Геометрия

Все расчёты в **миллиметрах**. Canvas масштабируется отдельно (zoom/pan).

## Модули

- `polygon.ts` — площадь, самопересечение, offset, шаблоны комнат
- `layout.ts` — сетка модулей, clipping через `martinez-polygon-clipping`

## Классификация модулей

Пересечение прямоугольника модуля с рабочим polygon:
- площадь пересечения ≈ площадь модуля → `full`
- площадь > 0 → `cut`
- иначе → `outside` (не отображается)

## Offset

Параллельный сдвиг рёбер внутрь с пересечением линий. При неудаче — сообщение пользователю.

## Шаблоны

- Прямоугольник
- Г-образная
- П-образная
- С нишей

## Оптимизация

`optimizeLayout()` перебирает смещения X/Y в пределах размера модуля, минимизируя количество подрезок.
