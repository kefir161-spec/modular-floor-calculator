# Известные ограничения

1. **Единица цены** — в XML не указана; итоговая стоимость не рассчитывается
2. **Артикул** — отсутствует в фиде; показывается только технический offer id
3. **Упаковки** — округление до упаковки не реализовано
4. **Обрезки** — повторное использование остатков не поддерживается
5. **Диагональная раскладка** — не реализована
6. **Игровые коврики** — исключены до решения заказчика
7. **Offset** — на очень сложных контурах может не построиться
8. **Оптимизация** — грубый перебор, для больших помещений может быть медленной
9. **Автосохранение** — сохраняется при ручном сохранении; полный autosave на каждое изменение не включён
10. **Изображения в PDF** — зависят от CORS политики plastfactor.com
